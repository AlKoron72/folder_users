# Streamlit Demo Project

This is a simple Python project using Streamlit.

## Getting Started

1. Install dependencies:
   
   ```powershell
   pip install -r requirements.txt
   ```
   
2. Run the app:
   
   ```powershell
   streamlit run app.py
   ```
